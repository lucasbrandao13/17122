Loja Atlético MG

Objetivo do Sistema
Este projeto tem como objetivo simular um site de vendas de camisas do Atlético Mineiro
O sistema apresenta:
Página inicial com produtos em destaque
Página de listagem de produtos
Simulação de carrinho de compras
Página de contato
Tudo foi desenvolvido de forma simples, utilizando apenas HTML e CSS

Tecnologias Utilizadas:
HTML E CSS
